集合类型

集合是由具有某些共同特征的元素构成的一个整体。在pascal中，一个集合是由具有同一有序类型的一组数据元素所组成，这一有序类型称为该集合的基类型

集合的定义

`var
    Vset1：set1；
    Vset2：set of byte；`

1、集合类型的所有成员必须是同一有序类型且最多只能含256个成员；
2、集合类型成员具有唯一性，即同一集合类型中不允许含有相同的成员；
3、集合类型的值域由BaseType的值域决定

标准的定义方式

`Type
    Set1 = set of byte;
    TMySet = set of Char;
    TMySet1 = set of 'a'..'z'; {集合&子界混合}
 var
    TMyset2 = set of 1..2;
    MySet: TMySet;`

byte是 基类型

实例

Type

Set1 = set of byte;         //set1的成员为byte型，值域为0至255，共255整型值

Set2 = set of 1..9;        //set2的成员为子界型，值域为1至9，共9整型值

Set3 = set of（red，green，blue）;        //set3的成员为枚举型，值域为3个标识符

合法的用法

type
  number = set of 0..9;
  ch      = set of char;
  days      = (sun,mon,tue,wed,thu,fri,sat);
var
  s:number;
  c:ch;
  week:days;

等同于

var 
  s:set of 0..9；                               //子界    
  c:set of char；
  week: (sun ,mon ,tue ,wed ,thu ,fri ,sat); //枚举

集合的值放在方括号中，中间用逗号隔开。

集合中没有任何元素，这样的集合称为空集。

集合中连续元素的值可以使用子界型的表示方法。

集合的值与集合内元素的出现次序无关。

集合中的同一元素重复出现对集合的值没有影响。

集合不能进行算数运算，集合是无序的，不能使用ord、pred（前一个元素）、succ（后一个元素）等函数。

赋值运算，只能通过赋值语句给集合变量赋值，不能通过读语句赋值，也不能通过写语句直接输出集合变量的值。

子界 (sub-boundaries)

type 子界类型 标识符 = 常量1…常量2

常量1称为子界的下界，常量2称为子界的上界

##

要求上界和下界为有序的数据类型 整数、字符、枚举

子界的上界大于或者等于下界

为了防止出现越界 

type 
    TSubBound=1..10；


type 
    age = 1..100;

    letter = 'a'..'z';


type 
    TsubBoundChar = 'a'..'z';
