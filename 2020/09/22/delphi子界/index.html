子界 (sub-boundaries)

type 子界类型 标识符 = 常量1…常量2

常量1称为子界的下界，常量2称为子界的上界

##

要求上界和下界为有序的数据类型 整数、字符、枚举

子界的上界大于或者等于下界

为了防止出现越界 

type 
    TSubBound=1..10；


type 
    age = 1..100;

    letter = 'a'..'z';


type 
    TsubBoundChar = 'a'..'z';
